## Instructions
1. リファクタリングや複雑なロジック作業の詳細なタスクと課題を `REFACTOR_QUEUE.md` に追記する。
2. 対象ファイルの枠組みを作成、または該当箇所に Claude 向けの日本語の指示コメントを記述して準備をする。
3. `notify_user` ツールを使用し、ユーザーに対して以下の明確なアクションを依頼してハンドオフ完了（あなた自身は待機状態）とする。
   - 例:「Claude 拡張機能を開き、入力欄に `@REFACTOR_QUEUE.md の最新タスクに従って実装してください` と送信して実行してください」
4. ユーザーからの作業完了通知（Accept等）を受け取った後、あなた（Antigravity）が該当ファイルを読み込み、検証および微細な修正を行う。
5. 作業と検証が完全に終了した後、あなたが `REFACTOR_QUEUE.md` のステータスを DONE に更新する。
