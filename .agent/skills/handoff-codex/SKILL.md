## Instructions
1. 対象ファイルの枠組みを作成し、内部にCodex向けの日本語の指示コメントを記述する。
2. `notify_user` ツールを使用し、ユーザーに対して以下の明確なアクションを依頼してハンドオフ完了（あなた自身は待機状態）とする。
   - 例:「Codex拡張機能を開き、入力欄に `@ファイル名 コメントに従って実装してください` と送信して実行してください」
3. ユーザーからの完了通知を受け取った後、あなた（Antigravity）が該当ファイルを読み込み、検証（Linter/型チェック等）および微細な修正を行う。
